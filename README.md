# Mini SOC Project

## Overview
Docker 기반 Mini SOC(Security Operation Center) 프로젝트

## Components
- Nginx
- Web Server
- MariaDB
- Wazuh
- Graylog
- Suricata

## Features
- 중앙 집중 로그 수집
- Wazuh 기반 보안 이벤트 탐지
- Graylog 로그 분석
- Suricata 네트워크 침입 탐지
- Discord 알림 연동
- AWS EC2 배포 테스트

## Technologies
- Linux
- Docker
- Docker Compose
- Nginx
- MariaDB
- Wazuh
- Graylog
- Suricata
- AWS EC2
  
## Key Experience
- Docker 기반 다중 서비스 구축
- Wazuh Agent 중앙 관리
- MariaDB 로그 탐지 룰 작성
- Graylog Dashboard 구성
- Suricata 탐지 및 차단 테스트
- AWS EC2 배포 경험
- 장애 복구 및 백업 자동화

## Project Goal
온프레미스 및 클라우드 환경에서 보안 모니터링 및 로그 분석 환경 구축

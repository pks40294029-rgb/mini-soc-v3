## 📅 프로젝트 기간

## 💡 프로젝트 개요
**AERO-SPACE** 부품 설계 데이터의 **기밀성 및 무결성 보장**을 목표로, 폐쇄망 기반의 차세대 보안 인프라를 구축하고 실시간 관제 시스템을 구현합니다.

---

## 🛠 Tech Stack
| 구분 | 기술/도구 | 용도 |
| :--- | :--- | :--- |
| **OS** | Ubuntu 24.04 LTS | 서버 인프라 |
| **🛡️ 방화벽** | OPNsense | 네트워크 망 분리 및 제어 |
| **🔍 IDS/IDP** | Snort | 패킷 기반 침입 탐지 |
| **📊 SIEM** | Wazuh | 로그 중앙 집중 관제 |
| **☁️ 웹 서비스** | Nextcloud | 데이터 보안 저장소 |
